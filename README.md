# How I Feel

On this website people can post about how they feel and can discuss with others about it. All posts are public. Tested with the latest PHP7.

![website preview](/preview.png)

### Features
- Users can pick from a list of feelings or write their own
- Leave a comment and talk about how someone else is feeling
- Post as anonymous or create and account and login

### Known Issues
- Login/Register is unstable and might have security problems
- Leaving comments is possible, but they are not displayed
- Emoticons are not properly implemented yet
